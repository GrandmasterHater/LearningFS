Эталон и решение отличаются в способе реализации функции вычисления факториала, но в остальном похожи.

### 49.5.3
Эталон и решение отличаются по способу решения. Эталон использует вычисления для выстраивания последовательности - если n чётное то оставляем n, если же n нечётное, 
то вычисляем по формуле `(-1) * (n + 1)/2` которая даёт положительную пару предыдущему отрицательному числу. Решение же использует построение последовательности
последовательностей пар значений с последующей развёрткой в общую последовательность через Seq.concat.
