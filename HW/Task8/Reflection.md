Решение первого задание совпадает с эталоном в логической части, однако отличается в записи уравнения - в эталоне используется x*(1 + n/100).
Если провести преобразования из решения, получится то же самое уравнение: 
x + (x*n) / 100 = (100*x)/100 + (x*n)/100 = x*(100/100 + n/100) = x*(1 + n/100)

vat. Для этого нужно сделать так, чтобы остался только множитель который преобразовывает число x на n % (Для уравнения из эталона этот множитель лучше виден).
Если вызвать vat n x c x=1, то функция вернёт тот самый множитель для заданного n. Таким образом разделив x на этот множитель получаем исходное
число.
