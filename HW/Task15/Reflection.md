### 40.4 revrev
Эталон и решение отличаются. Эталон использует паттерн матчинг для задания всей логики переворачивания как исходного списка, так и вложенных. Решение же определяет 2 функции -
rev, для переворачивания списка и map для применения функции к каждому аргументу списка. Сама логика состоит из последовательной передачи списка на вход каррированной функции
map rev, чтобы перевернуть вложенные списки, а затем результат передаётся в функцию rev для переворачивания исходного списка. Оба варианта просты, понятны и выполняют
поставленную задачу.
